# vampire-restaurant

## ACKNOWLEDGEMENT

This website was based off a menu that my friends made for our Halloween party - we all had a great time then, so I wanted to make this project into a keepsake of that party :)

Initially, I've tried to extract the SVG images from the PDF menu using <a href="https://www.pdf2svgsnipper.com/">this tool</a> and I removed their backgrounds using <a href="https://deeditor.com/">this site</a>, but the resulting images were low in quality. So instead, I went to Canva, where the initial menu was made, downloaded image assets from there, converted them to SVG format using <a href="https://pngtosvg.com/">this site</a> and optimized/cleaned them up using <a href="https://iconostack.com/svg-code-to-file/">this tool</a>.

Afterwards, I've used <a href="https://codepen.io/sosuke/pen/Pjoqqp">this codepen solution</a> to get the right filter values for bat color variation
